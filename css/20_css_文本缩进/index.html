王健林当时表示，自己首先是看好沈阳的未来发展；其次，沈阳投资营商环境发生深刻改变，现在市委市政府大力招商引资，服务热情周到，效率极大提升。沈阳市政府和万达集团的战略合作从谈判到签约，只用时一个月，双方约定今年三季度开工建设。再者，万达已在沈阳持续投资18年，开业6个项目，在沈阳的全部投资项目都赚了钱，在沈阳运营的商业和酒店效益也不错，这些都让万达坚定了投资沈阳的决心，而且王健林坚信沈阳会在东北率先振兴。

# html 简介

- html   HyperText Markup Language   超文本标记语言  
- 描述网页的结构

- 什么是文本？

  - 我们平常在记事本里写的文字 

  - 写一个网页

- 什么是超文本？

  - 不仅仅是文字，包括图片，视频等

- 什么是标记？

  - 用标记把内容包括起来 ，标记也叫标签，开始标签（也叫起始标签），结束标签（也叫闭合标签）
  - 试着把网页写的内容用h1标签包括起来

- 写一个基本网页的框架

  ```javascript
  <html> 根标签,只用一个，最外层 
  <head>  头部
  <body>  身体 
  <title> 标题    
  ```

# html_不常用标签

- strong  加粗字体
- em   斜体
- cite 引文
- del 删除线
- ins，u 下划线
- sup，sub 上标字，下标字
- tt 打字机
- big,small 大字体，小字体
- blockquote 引用

```javascript
<div>正常</div>
<strong>加粗</strong>
<cite>引用</cite>
<del>删除</del>

<ins>下划线</ins>
<u>下划线</u>

<sup>上标字</sup>
<sub>下标字</sub>

<tt>打字机</tt>
<big>大写的</big>
<small>小写的</small>

<blockquote>引用</blockquote>
```
